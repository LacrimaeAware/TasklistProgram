Getting started
• Add a task: fill Title and (optional) Due/Priority/Repeat, then click Add.
• Edit a task: double-click a row.
• Mark done: select task(s) and click Mark Done.
• Delete or restore: use Delete/Restore or the right-click menu.

Fields explained
• Title: a short label for the task.
• Due: accepts date-only or date+time (see examples below).
• Priority: H/M/L/D/Misc (D = daily habit).
• Repeat: none, daily, weekdays, weekly, monthly.
• Notes: free text, useful for details.
• Group: optional label used for grouping in the list.

Due date formats (examples)
• 2024-10-05
• 2024-10-05 14:00
• 10/05
• 10/05 14:00
• 14:00 or 1400 (today at 2:00 PM)
• midnight (today at 23:59)
• +2d +3h (relative)

Tips
• Use Group view to collapse tasks by group.
• Search filters titles and notes.
• Group view and filter selections are saved between runs.
